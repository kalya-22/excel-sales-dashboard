Excel Sales Dashboard

Project Overview

This project analyzes Superstore sales data using Microsoft Excel. The objective was to clean the dataset, perform data analysis using Pivot Tables, and create an interactive dashboard for business insights.

Dashboard Preview

![Excel Sales Dashboard](dashboard%20sales.png)

Objectives

* Analyze Total Sales and Total Profit
* Identify top-performing product categories
* Track monthly sales trends
* Analyze top customers
* Enable interactive filtering using slicers

Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

Dashboard Features

KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Total Customers

Visualizations

* Sales by Region
* Profit by Category
* Monthly Sales Trend
* Top 10 Customers

Interactive Filters

* Region
* Category
* Segment
* Year

Key Insights

* The West region generated the highest sales.
* Technology was the most profitable category.
* Sales increased significantly during late 2017.
* A small number of customers contributed a large share of revenue.
